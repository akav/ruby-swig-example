# ruby-swig-example

Examples of writing ruby bindings for C/C++ library using SWIG.

* ext/callback (using the director feature of SWIG for callback)
* ext/conv
* ext/enum
* ext/extend
* ext/mod_narray (converting `Numo::NArray` to `std::vector<double>`)
* ext/multia
* ext/multib
* ext/sharedptr
* ext/stl
* ext/template
* ext/typemap
* ext/uniqueptr
* ext/unorderedmap
